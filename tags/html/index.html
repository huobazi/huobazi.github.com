ASP.NET Asp.net Javascript HTML

在net_lover的文章里找到了答案.

http://www.csdn.net/Develop/Read_Article.asp?Id=4405

代码如下：
<HTML>
<HEAD>
</HEAD>
<BODY>
<DIV ID=“oDiv”>TEST—TEST—TEST 您可以在这里编辑文字！</DIV>
<SCRIPT>
<!–
oDiv.contentEditable = true;
// –>
</SCRIPT>
</BODY>
</HTML>
